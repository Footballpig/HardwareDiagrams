Hardware Diagrams


Hardware is the machinery inside any apparatus. In this example of hardware units, the machinery is inside a desktop. Some machinery isn't needed to run, but it will run better if the apparatus has these parts.

Desktop The Desktop is supposed to hold and protect all of the internal pieces of hardware.
Motherboard The Motherboard is supposed to be the parent of the computer. It comands everything and plugs everything together.
HardDrive The HardDrive is supposed to store all of the information on the machine.
RAM The RAM stands for Random Access Memory. It temporary stores information.
VideoCard The VideoCard is an add on to the machine. It is used to add sound quality to the device.
